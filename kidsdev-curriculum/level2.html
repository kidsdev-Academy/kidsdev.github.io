Detailed Syllabus

From Blank Page to Live Site

We teach the exact tools used by professionals at Google and Facebook, adapted for young learners.

1

The Structure (HTML5)

Weeks 1-3

Students learn that a website is like a house, and HTML is the frame. We focus on writing "Semantic HTML"‚Äîcode that is meaningful, accessible to screen readers, and structured correctly for search engines (SEO).

Semantic Tags: Using <header>, <nav>, and <main> instead of just div.
Client vs Server: How browsers render code from a remote server.
Rich Media: Embedding Audio, Video, and Iframes correctly.
2

The Style (CSS3)

Weeks 4-6

Coding meets Art. We introduce the "Cascading Style Sheets" language. Students learn how to select HTML elements and apply design rules, understanding inheritance and specificity (why some rules override others).

Advanced Selectors: Targeting specific classes, IDs, and states (:hover).
Visual Design: Working with Google Fonts, Hex Codes, and Gradients.
The Cascade: Understanding how style rules flow down to elements.
3

Advanced Layouts

Weeks 7-9

Moving elements around the page is the hardest part of web design. We master the Box Model (Margins vs Padding) and use modern systems like Flexbox to create layouts that work on Phones, Tablets, and Laptops.

Box Model: The physics of web layout (Margin, Border, Padding).
Flexbox: Creating flexible rows and columns for navigation bars.
Responsiveness: Using Media Queries to adapt sites for mobile screens.
4

Capstone & Deployment

Weeks 10-12

A website isn't finished until it's online. Students learn the professional workflow: cleaning up code, optimizing images for speed, and deploying via GitHub Pages. They finish with a live URL they can share.

GitHub Deployment: Publishing code to a live server.
Forms: Building working "Contact Me" sections.
Debugging: Using Chrome DevTools to fix layout issues.
