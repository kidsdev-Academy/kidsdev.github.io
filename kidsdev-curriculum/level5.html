Detailed Syllabus

The Modern Web Stack

We teach the exact technology stack used by companies like Facebook, Netflix, and Uber.

1

Advanced JavaScript (ES6+)

Weeks 1-4

Before frameworks, you must master the language. We dive deep into modern JavaScript, focusing on asynchronous operations (getting data from the internet) and functional programming.

Async/Await: Handling API requests and Promises.
ES6 Syntax: Arrow functions, Destructuring, Spread operators.
DOM Manipulation: Dynamic content updates without reloading.
2

Frontend Engineering (React)

Weeks 5-8

We switch to industry-standard tools. Students learn React.js to build "Single Page Applications" (SPAs)‚Äîwebsites that feel as fast and smooth as mobile apps.

Components: Building reusable UI blocks (Navbars, Cards).
State Management: Using Hooks (useState, useEffect) for data.
Routing: Creating multi-page experiences in React.
3

Backend & API (Node.js)

Weeks 9-12

The logic behind the scenes. Students build their own Server and API using Node.js and Express, learning how data moves securely across the internet.

REST APIs: Creating endpoints (GET, POST, DELETE).
Express Server: Handling web requests and routing.
Authentication: Basic user login and security concepts.
4

Database & Deployment

Weeks 13-16

Data persistence. We connect our app to a Database (MongoDB/Firebase) so user data is saved forever. Finally, we deploy the app to the cloud using Vercel.

NoSQL Databases: Storing user profiles and app data.
Cloud Deployment: Going live on Vercel or Netlify.
Git Workflow: Professional version control.
