Detailed Syllabus

Engineering, Not Just Coding

We focus on clean code, efficiency, and logic. This curriculum is designed to build the mindset of a Senior Developer.

1

The Grammar of Code

Weeks 1-3

Just like learning English or French, coding has grammar (Syntax). We start by installing professional tools (Thonny IDE) and learning how to speak to the computer correctly using Python.

Variables & Data Types: Storing Strings, Integers, and Floats.
Input/Output: Creating interactive text-based programs.
Operators: Performing complex math and logic calculations.
2

Control Flow & Logic

Weeks 4-6

Programs need to make decisions. "If the password is correct, grant access." We teach students how to write programs that can think and branch into different paths based on data.

Conditionals: Mastering If, Elif, and Else statements.
Boolean Logic: True/False and comparison operators.
Debugging: Reading error messages and fixing logic flaws.
3

Data Structures

Weeks 7-9

Professional software handles massive amounts of data. Students learn how to organize information efficiently using Lists (Arrays) and Dictionaries (Key-Value pairs), the backbone of modern apps.

Lists & Indexing: Storing and retrieving sequences of data.
Dictionaries: Creating complex data models (e.g., User Profiles).
Loops: Using For and While loops to process data automatically.
4

Algorithms & Functions

Weeks 10-12

Efficiency matters. We learn to write reusable code using Functions and import external libraries (like Random or Math) to build a powerful final application.

Functions: Defining inputs (Parameters) and outputs (Return).
Modules: Using Python's vast library ecosystem.
Capstone: Building a Data Analysis or Logic Game app.
