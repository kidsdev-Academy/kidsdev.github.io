Detailed Syllabus

Red Team vs. Blue Team

We teach a balanced curriculum of Offensive Security (Attacking to find flaws) and Defensive Security (Securing systems).

1

The Hacker's Lab (Linux)

Weeks 1-3

We leave Windows behind. Students learn to install and manage Kali Linux in a Virtual Machine. They master the Command Line Interface (CLI), the primary tool of cyber professionals.

Kali Linux: The OS of security professionals.
BASH Scripting: Automating security tasks.
Virtualization: Safe sandboxing with VirtualBox.
2

Networking & Reconnaissance

Weeks 4-6

You can't hack what you can't find. We deep dive into how the internet works (IPs, Ports, DNS) and learn to map out networks using professional scanning tools.

OSI Model: How data moves (TCP/UDP Handshakes).
Nmap Scanning: Finding open ports and devices.
Wireshark: Analyzing live network traffic packets.
3

Offensive Security (Web)

Weeks 7-9

Ethical Hacking. Students practice on a dedicated "Vulnerable Web App" (DVWA) to understand common flaws like SQL Injection and Cross-Site Scripting (XSS).

OWASP Top 10: The most critical web vulnerabilities.
SQL Injection: How databases get compromised.
Social Engineering: Understanding phishing and human error.
4

Defense & Reporting

Weeks 10-12

The goal of an Ethical Hacker is to fix, not break. Students learn to patch vulnerabilities, configure Firewalls, and write professional Security Audit Reports.

System Hardening: Securing Linux servers and ports.
Reporting: Writing clear vulnerability assessments.
Cryptography: Basics of encryption and hashing.
