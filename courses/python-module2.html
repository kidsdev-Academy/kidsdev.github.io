Variables & Data Types

In this module, we will learn how to store information (Variables) and the different kinds of information Python understands (Data Types).

2.1 Variables & Naming Conventions

Think of a Variable like a labelled box. You can put data inside it, and when you want that data back, you just look for the label.

my_score = 100
player_name = "Mario"

print(player_name) # Prints: Mario

Naming Rules (Snake Case)

Python is picky about names. Here are the rules:

  • ‚úÖ Do: Start with a letter (a1 is okay).
  • ‚ùå Don't: Use spaces (my score is wrong).
  • ‚ùå Don't: Start with a number (1player is wrong).
2.2 Numeric & Text Data Types

Computers categorize data. In Python, the three most common types are Strings, Integers, and Floats.

String (str)

Text inside quotes.

"Hello"

Integer (int)

Whole numbers.

50

Float

Decimal numbers.

3.14

You can check the type of any data using the type() command:

print(type("Hello")) # <class 'str'>
print(type(10)) # <class 'int'>
2.3 Boolean Values

A Boolean is the simplest data type. It can only be one of two things: True or False.

Think of a light switch. It is either ON (True) or OFF (False). We use Booleans for logic and decision making.

is_game_over = False
has_key = True

print(is_game_over)
üìù Note: In Python, True and False must start with a Capital Letter!
2.4 Type Conversion (Casting)

Sometimes we need to change data from one type to another. This is called Casting.

For example, the input() function always gives us a String. If we want to do math, we must convert it to an Integer.

# This is a string "5"
age_text = "5"

# Convert it to a number (5)
age_number = int(age_text)

# Now we can do math!
print(age_number + 1)
  • int() - Converts to whole number.
  • float() - Converts to decimal number.
  • str() - Converts to text.
