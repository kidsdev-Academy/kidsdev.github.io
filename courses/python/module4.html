The Fork in the Road

Imagine you are walking down a path and it splits in two. If you go Left, you find treasure. If you go Right, you find a dragon.

Computers make these decisions using If Statements. They check if something is True or False, and then decide what code to run next.

If, Elif, Else

Think of it like a video game menu:

button = "Start" if button == "Start": print("Loading Game...") elif button == "Options": print("Opening Settings...") else: print("I don't know that button.")
  • if: The first check.
  • elif: "Else If" - checks another option if the first one failed.
  • else: The backup plan if nothing else matched.
