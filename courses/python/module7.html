Python Lists

Imagine creating a variable for every student in your class: student1, student2, student3... that would be exhausting!

Instead, we use a List. It's one single variable that holds many items, like a shopping bag.

Creating a List

We use square brackets [] to make a list.

# A list of strings backpack = ["Sword", "Potion", "Map"] # A list of numbers scores = [10, 50, 100, 25] print(backpack)

The Index (Position)

⚠️ Computers Start at 0

The first item is always at index 0. The second item is at index 1.

items = ["Apple", "Banana", "Cherry"] print(items[0]) # Prints "Apple" print(items[1]) # Prints "Banana"
