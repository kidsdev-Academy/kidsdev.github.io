Intro to Data Science

Forget games for a moment. Let's use Python for its real-world superpower: analyzing massive amounts of data using a library called Pandas.

Phase 1: The Setup (Pandas)

Unlike Pygame, Pandas is used in the terminal. First, we need to install it and import it. We usually nickname it `pd`.

# In your terminal first: # pip install pandas # In your Python file: import pandas as pd # Create some fake data data = { 'Student': ['Tola', 'David', 'Sarah', 'Chris'], 'Score': [85, 92, 78, 95], 'Hours_Studied': [5, 8, 3, 9] }
The Goal:

We are creating a "Dictionary" of data that looks like a spreadsheet, ready to be loaded into Pandas.

Phase 2: The DataFrame

The core of Pandas is the DataFrame. It converts our dictionary into a neat table that Python understands.

# Load data into a DataFrame df = pd.DataFrame(data) # Print the whole table print(df)
Terminal Output:
Student Score Hours_Studied 0 Tola 85 5 1 David 92 8 2 Sarah 78 3 3 Chris 95 9

Phase 3: Analyzing the Data

Now for the magic. We can ask Python questions about the data instantly, without counting manually.

# Calculate the Average Score avg_score = df['Score'].mean() print(f"Average Score: {avg_score}") # Find the Highest Score max_score = df['Score'].max() print(f"Highest Score: {max_score}") # Filter: Who studied more than 6 hours? hard_workers = df[df['Hours_Studied'] > 6] print("\nHard Workers:\n", hard_workers)
Terminal Output:
Average Score: 87.5 Highest Score: 95 Hard Workers: Student Score Hours_Studied 1 David 92 8 3 Chris 95 9
