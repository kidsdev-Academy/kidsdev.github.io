Module 5

While Loops

Imagine you have a whole pizza. You eat one slice. Then another. You keep eating WHILE you are still hungry. Once you are full, you stop.

Repeating Code

A while loop runs the same block of code over and over as long as the condition is True.

slices = 8 while slices > 0: print(f"Eating slice! {slices} left.") slices = slices - 1 print("All done!")
⚠️ The Infinite Loop

If you forget to subtract a slice (slices - 1), the computer will think there are 8 slices forever and never stop running. This crashes the computer!

Create a variable countdown = 10. Write a while loop that prints the number and counts down until 0. Then print "Blast off!".
