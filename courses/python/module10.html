Intro to Pygame

Welcome to Game Development! We are going to use a special library called Pygame. Think of it like a box of tools that helps Python draw shapes, play sounds, and read keyboard clicks.

The Infinite Loop

A video game is just a loop that runs 60 times a second. It checks for inputs, updates positions, and draws the screen.

import pygame pygame.init() # 1. Create the Window screen = pygame.display.set_mode((800, 600)) # 2. The Game Loop running = True while running: for event in pygame.event.get(): if event.type == pygame.QUIT: running = False # Draw a red rectangle pygame.draw.rect(screen, (255,0,0), (100, 100, 50, 50)) pygame.display.flip()
