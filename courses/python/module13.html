Phase 2: Movement & Direction

The snake needs to keep moving automatically. We use variables like `x_change` and `y_change` to remember which direction it's heading. Pressing an arrow key changes these variables.

# Change in position x_change = 0 y_change = 0 # Inside Loop, check keys: if event.key == pygame.K_LEFT: x_change = -snake_block y_change = 0 elif event.key == pygame.K_RIGHT: x_change = snake_block y_change = 0 # Apply movement automatically every frame snake_x += x_change snake_y += y_change
