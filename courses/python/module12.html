Collisions

If you run into a wall in a video game, you stop. If you run into a coin, you collect it. This is called Collision Detection, and it's how games know when things touch.

Rectangles Touching

Pygame makes this easy. We create invisible boxes called "Rects" around our sprites, and ask Pygame if they are overlapping.

player_rect = pygame.Rect(x, y, 50, 50) enemy_rect = pygame.Rect(300, 300, 50, 50) # The magic command if player_rect.colliderect(enemy_rect): print("GAME OVER!") running = False
