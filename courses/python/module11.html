Module 11

Moving Sprites

A game isn't fun if you can't control it! Today we will learn how to hook up your Arrow Keys to move a character around the screen.

Coordinates (X and Y)

The screen is a grid. To move Right, we increase X. To move Down, we increase Y.

# Starting position x = 100 y = 100 # Inside the game loop... keys = pygame.key.get_pressed() if keys[pygame.K_RIGHT]: x = x + 5 # Move Right if keys[pygame.K_LEFT]: x = x - 5 # Move Left # Draw player at new position pygame.draw.rect(screen, (0,255,0), (x, y, 50, 50))
