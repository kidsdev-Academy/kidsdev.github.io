The Magic of Memory

Imagine you are moving houses. You have a lot of stuff, so you put your toys into a box and label it "Toys". Later, when you need a toy, you don't search the whole house; you just look for the box labeled "Toys".

In coding, we do the exact same thing! We call these boxes Variables. A variable is a container where we store data so the computer can find it later.

Creating Your First Variable

In Python, creating a variable is super easy. You just give it a name, use the equals sign (=), and give it a value.

# Creating a variable named 'player_name' player_name = "Tola" # Creating a variable named 'score' score = 500 print(player_name) # The computer looks inside the box and prints: Tola

Naming Rules

  • No spaces allowed! Use underscores: my_score
  • Cannot start with a number (1up is bad).
  • Case matters! Apple and apple are different.
