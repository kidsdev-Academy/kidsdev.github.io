Module 2

Data Types & Math

Computers are picky! They need to know if you are giving them a word or a number. If you try to do math with words, the computer gets confused.

The Big Three

String (str)

Text inside quotes. Even numbers in quotes count as text!

name = "Tola"
Integer (int)

Whole numbers. No decimals.

lives = 3
Float

Numbers with decimals (like money).

price = 19.99

Doing Math

score = 10 bonus = 5 # Addition total = score + bonus # Result: 15 # Multiplication double_score = score * 2 # Result: 20 # Division half_score = score / 2 # Result: 5.0
