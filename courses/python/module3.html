Talking to Computers

So far, your code runs and finishes. But what if you want to ask the player for their name? Or ask them which door to open? We use the input() function!

Asking Questions

The input() command stops the program and waits for the user to type something.

print("What is your name?") name = input() print(f"Nice to meet you, {name}!")

The String Problem

⚠️ Careful!

input() always returns text (String). Even if you type a number!

age = input("How old are you? ") # This will crash if you don't convert it! # next_year = age + 1 <-- ERROR # Do this instead: next_year = int(age) + 1 print(f"Next year you will be {next_year}")
