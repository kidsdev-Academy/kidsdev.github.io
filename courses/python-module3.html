Module 3

Loops & Repetition

Programmers are lazy! We don't like writing code twice. In this module, we learn how to make the computer repeat things for us automatically.

3.1 While Loops

A while loop is like a hungry person eating. It keeps running as long as a condition is True.

[Image of while loop flowchart]
energy = 5

while energy > 0:
  print("Running... üèÉ")

print("I am tired now!")

3.2 For Loops & Ranges

A for loop is different. It is used when you know exactly how many times you want to repeat something.

We use the range() function to count.

# This will count from 0 to 4
for i in range(5):
  print(i)

Note: Computers start counting at 0, not 1!

3.3 Lists (The Backpack)

So far, a variable can only hold one thing. But what if you want to store a list of friends, or high scores? We use a List (also called an Array).

Think of a List like a Backpack where you can put many items inside.

fruits = ["Apple", "Banana", "Cherry"]

# Let's print everything in the backpack
for item in fruits:
  print("I love " + item)
3.4 Project: Times Table Generator

Let's build a tool to help with your math homework! This program asks for a number and prints the multiplication table.

# 1. Ask for a number
number = int(input("Which multiplication table? "))

# 2. Loop from 1 to 12
for i in range(1, 13):
  answer = number * i
  print(f{number} x {i} = {answer}")
