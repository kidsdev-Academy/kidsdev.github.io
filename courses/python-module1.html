Module 1

Introduction to Python

Welcome to the world of coding! In this module, we will learn how to speak the language of computers, set up our tools, and write our very first program.

1.1 What is Python?

Imagine you want to talk to a person from France, but you only speak English. You would need a translator, right?

Python is a translator for computers. Computers only understand zeros and ones (010101), which is very hard for humans. Python lets us write instructions in simple English, and then it translates them for the computer.

Why is it great for kids?

  • It's Readable: The code looks almost like reading a normal sentence.
  • It's Powerful: Used by NASA, YouTube, and Instagram.
  • It's Versatile: Make games, websites, or AI.
1.2 Installing Python & IDLE

Before we code, we need the software. Follow these steps carefully:

  1. Go to python.org.
  2. Click Downloads and get the latest version.
  3. VERY IMPORTANT: Check the box "Add Python to PATH" before installing!
  4. Click "Install Now".

What is IDLE?

Along with Python, you installed IDLE. Think of IDLE as your "Sketchbook". It is a simple place where we can write code and run it immediately to see what happens.

1.3 Your First Code (Hello World)

Let's use the print() function. It tells the computer: "Show this text on the screen."

print("Hello, World!")

The Rules (Syntax):

  • You must use parentheses ( ) after the word print.
  • You must put text inside quotation marks " ".
Fun Challenge:

Try writing a second line of code to print your own name! Example: print("My name is Alex")

1.4 Variables (The Box)

In programming, we need to save information to use later. We use Variables to do this.

The Box Analogy

Imagine you have an empty cardboard box. You can write a label on the outside (the Name) and put something inside (the Value).

  • Label: score
  • Value inside: 100

In Python, we use the = sign. It doesn't mean "equals" like math; it means "assign to" or "put inside".

# We create a box called 'player_name' and put 'Sarah' inside it
player_name = "Sarah"

# We create a box called 'score' and put 50 inside it
score = 50

# Ask Python to show what is in the box
print(player_name)
print(score)
1.5 Basic Data Types

Computers organize data into different types. Just like you sort your laundry (Socks vs Shirts), Python sorts data.

  • 1. Strings (str)
    Text inside quotes.
    "Hello", "123 Main St"
  • 2. Integers (int)
    Whole numbers (no decimals). Like a whole pizza.
    5, 100, -20
  • 3. Floats (float)
    Numbers with decimals. Like a slice of pizza.
    2.5, 3.14
