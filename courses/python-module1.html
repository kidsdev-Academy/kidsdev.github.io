1.1 What is Python & Why learn it?

Imagine you want to talk to a person from France, but you only speak English. You would need a translator, right?

Python is a translator for computers. Computers only understand zeros and ones (010101), which is very hard for humans to read. Python lets us write instructions in simple English, and then it translates those instructions so the computer understands.

Why is Python the best language for kids?

  • It's Readable: Python code looks almost like reading a normal sentence.
  • It's Powerful: It's used to build real things like YouTube, Instagram, and even by NASA scientists!
  • It's Versatile: You can use it to make games, build websites, or create Artificial Intelligence (AI).
1.2 Installing Python & IDLE

Before we can code, we need to install the software on your computer. Follow these steps carefully:

  1. Go to the official website: python.org.
  2. Click on Downloads and download the latest version for your computer.
  3. VERY IMPORTANT: When the installer opens, you will see a small checkbox at the bottom that says "Add Python to PATH". Check this box! If you don't, your computer might not find Python later.
  4. Click "Install Now" and wait for it to finish.

What is IDLE?

Along with Python, you installed a program called IDLE. Think of IDLE as your "Sketchbook." It is a simple place where we can write code and run it immediately to see what happens.

1.3 Your First Code (Hello World)

Let's write your first program! We are going to use a special command (function) called print().

The print() function tells the computer: "Show this text on the screen."

Try this in IDLE:

print("Hello, World!")

The Rules (Syntax):

  • You must use parentheses ( ) after the word print.
  • You must put your text inside quotation marks " ". This tells Python that "Hello World" is text, not a command.
üí° Fun Challenge: Try writing a second line of code to print your name!
Example: print("My name is Alex")
1.4 Understanding Variables (The Box)

In programming, we need to save information to use later. We use Variables to do this.

The Box Analogy

Imagine you have an empty cardboard box. You can write a label on the outside (the Name) and put something inside (the Value).

  • Label: score
  • Value inside: 100

In Python, we create a variable using the = sign. This is not "equals" like in math; it means "assign to".

# We create a box called 'player_name' and put 'Sarah' inside it
player_name = "Sarah"

# We create a box called 'score' and put 50 inside it
score = 50

# Now we can ask Python to show us what is in the box
print(player_name)
print(score)

When you run this, Python looks inside the box labeled player_name and prints "Sarah".

1.5 Basic Data Types (Strings & Numbers)

Computers organize data into different types. Just like you sort your laundry into "Socks," "Shirts," and "Pants," Python sorts data into types.

1. Strings (str)

A string is just text. It is a string of characters (letters) tied together. In Python, strings must be inside quotes.

  • "Hello"
  • "123 Main Street"

2. Integers (int)

An integer is a whole number. It does not have any decimal points. Think of it like a whole pizza.

  • 5
  • 100
  • -20

3. Floats (float)

A float is a number with a decimal point. Think of this like a slice of pizza (a fraction of a whole).

  • 2.5
  • 3.14
  • 99.9
üìù Remember: If you put a number in quotes like "5", Python treats it as a String (text), not a number!
