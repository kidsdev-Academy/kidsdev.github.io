IP Addresses & DNS

Every house has an address so the mailman can find it. Every computer on the internet has an address too! We call it an IP Address.

What does it look like?

192.168.1.1

4 numbers separated by dots.

Computers love numbers. But humans are bad at remembering them. Can you memorize Google's IP address? Probably not.

DNS: The Phonebook

DNS (Domain Name System) is the internet's contact list. It matches names to numbers.

1. You Type: www.roblox.com
2. DNS Finds: 128.116.119.4
