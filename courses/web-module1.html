The HTML Skeleton

Imagine you are building a robot. Before you paint it (CSS) or make it move (JavaScript), you need to build the metal frame. That frame is HTML.

HTML stands for HyperText Markup Language. It uses "tags" to tell the browser what is a heading, what is a paragraph, and what is an image.

The Basic Structure

Every website in the world starts with this exact code. It's like the bun of a burger—it holds everything together.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Website</title>
  </head>
  <body>

    <h1>Hello World!</h1>
    <p>This is my very first webpage.</p>

  </body>
</html>

Tags Explained

  • <html> The wrapper for your entire site.
  • <head> Information for the browser (like the tab title). Users don't see this directly.
  • <body> Everything the user sees (text, images, buttons) goes here.

Your Mission

1. Create a folder on your desktop named MyWebsite.
2. Open VS Code and create a file named index.html.
3. Copy the code above and save it.
4. Double-click the file to open it in Chrome!
