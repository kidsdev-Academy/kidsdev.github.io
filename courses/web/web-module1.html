Building Skeletons

Every website you visit—YouTube, Roblox, Google—is built using a language called HTML (HyperText Markup Language).

Think of HTML as the Skeleton of a website. It doesn't handle the colors (that's CSS) or the movement (that's JS), but it provides the structure that holds everything together.

Tags: The Bricks of the Web

HTML uses "Tags" to tell the browser what to show. A tag usually has an opening <tag> and a closing </tag>.

<!-- This is a Heading --> <h1>My Super Hero Page</h1> <!-- This is a Paragraph --> <p>Welcome to my fan club.</p> <!-- This is a Button --> <button>Join Now</button>

Essential Tags to Memorize

  • <h1>

    The biggest title on the page. Use only one per page!

  • <img>

    Shows a picture. It creates a window for an image file.

  • <a>

    The "Anchor" tag. This creates a link to another website.
