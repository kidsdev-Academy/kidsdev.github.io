Think of HTML as the Skeleton. It doesn't handle the style (that's CSS) or the brains (that's JS), but it holds everything together.

Tags: The Bricks

HTML uses "Tags" to tell the browser what to show. A tag usually has an opening <tag> and a closing </tag>.

<!-- This is a Heading --> <h1>My Super Hero Page</h1> <!-- This is a Paragraph --> <p>Welcome to my fan club.</p> <!-- This is a Button --> <button>Join Now</button>

Essential Tags to Memorize

<h1>

The biggest title. Only use one per page!

<img>

Shows a picture. It needs a src link.

<a>

Anchor tag. Creates a link to another site.
