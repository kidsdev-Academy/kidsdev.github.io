If HTML is the Skeleton, then CSS (Cascading Style Sheets) is the Costume.

Without CSS, every website would look like a boring Word document. CSS handles colors, fonts, spacing, and layout.

How it Works

CSS works by "Selecting" an HTML tag and then applying "Rules" to it.

/* This selects ALL <h1> tags */ h1 { color: red; font-size: 50px; text-align: center; } /* This selects ALL <p> tags */ p { color: blue; }

Essential Properties

color

Changes the text color. (e.g., red, #000000)

background-color

Changes the background color behind the text.

padding

Adds breathing room inside a box.
