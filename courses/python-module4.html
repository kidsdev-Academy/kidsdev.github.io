Module 4

Functions

Don't repeat yourself! Learn how to create your own "mini-programs" that you can use over and over again.

4.1 What is a Function?

Imagine you have a magic spell. Instead of reading the whole spell book every time, you just say "Abracadabra!" and it happens.

In Python, a Function is a block of code that only runs when you call it by name. We use the def keyword.

def say_hello():
  print("Hello, World!")

# Now we call the function to use it
say_hello()
say_hello()

This will print "Hello, World!" twice because we called it twice.

4.2 Parameters (Inputs)

Functions can accept data. These are called Parameters. Think of them like ingredients for a recipe.

def greet(name):
  print("Hello, " + name)

greet("Alice")
greet("Bob")
Result:
Hello, Alice
Hello, Bob
4.3 Return Values (Outputs)

Sometimes we want the function to give us an answer back, instead of just printing it. We use the return keyword.

def add_numbers(a, b):
  return a + b

answer = add_numbers(5, 10)
print(answer) # Prints 15
4.4 Project: Mini Calculator

Let's combine everything we've learned to make a simple calculator!

def add(x, y):
  return x + y

def subtract(x, y):
  return x - y

print("5 + 3 =", add(5, 3))
print(10 - 4 =", subtract(10, 4))
