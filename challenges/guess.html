Guess the Number Game

Can you build a program that thinks of a number between 1 and 100, and guides the user to guess it?

Computer Logic

  • Pick a random number.
  • Ask user for input.
  • If Guess > Number: Print "Too High!"
  • If Guess < Number: Print "Too Low!"
  • If Guess == Number: Print "You Win!"

> Guess a number: 50

Too High!

> Guess a number: 25

Too Low!

> Guess a number: 30

Correct! You won in 3 tries.

Starter Code

You'll need the `random` module for this.

game.py
import random

number = random.randint(1, 100)
tries = 0

while True:
  guess = int(input("Pick a number: "))
  tries += 1

  if guess == number:
    print("You won!")
    break
  elif guess > number:
    print("Too high!")
  else:
    print("Too low!")
