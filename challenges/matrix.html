The Matrix Rain Effect

Create the iconic "Digital Rain" using JavaScript. This is the perfect introduction to creative coding.

1 The Concept

We treat the screen as a grid of columns. Each column has a "drop" falling down it.

const columns = canvas.width / fontSize;
const drops = []; // Stores y-position of each drop

// Start all drops at y=1
for(let i=0; i<columns; i++) {
  drops[i] = 1;
}

2 The Fade Effect

To make the trails fade, we don't clear the screen completely. We draw a semi-transparent black box over it.

ctx.fillStyle = "rgba(0, 0, 0, 0.05)";
ctx.fillRect(0, 0, canvas.width, canvas.height);
