Design a Personal Card

Your mission, should you choose to accept it: Create a digital ID card for yourself using HTML & CSS.

Super Coder

Full Stack Developer

"I turn coffee into code and ideas into reality."

The Challenge Requirements

  • Use a div container for the card.
  • Add a round profile image (Use border-radius: 50%).
  • Add your Name (`h2`) and Role (`h4`).
  • Add a subtle box-shadow to make it pop off the screen.

Starter Code

Copy this into your editor to get started:

<div class="card">
  <img src="me.jpg" class="avatar">
  <h2>Your Name</h2>
  <p>Web Developer</p>
</div>
