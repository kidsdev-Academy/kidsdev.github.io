How Passwords are Stored

If a website saves your password as plain text (like "Secret123"), it is very bad! Instead, they turn it into a Hash.

Input: "Pizza"

Hash: 8d969eef6ecad3c29a3a629280e686cf


Input: "Pizza!" (Just one change)

Hash: 50f683e892c44234f9a727282121775a

A hash is a one-way street. You can turn "Pizza" into code, but you can't turn the code back into "Pizza". This keeps you safe!
