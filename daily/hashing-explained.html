How Passwords are Stored

If a website saves your password as plain text (like "Secret123"), it is a bad website! Instead, they turn it into a Hash.

The Magic Machine

A hash is a one-way street. You can turn "Pizza" into code, but you can't turn the code back into "Pizza". This keeps you safe even if the database is hacked.

Input

"Pizza"

Output (MD5 Hash)

8d969eef6ecad3c29a3a629280e686cf

Fun Fact

Even changing one letter completely changes the hash. "Pizza!" produces a totally different code than "Pizza".
