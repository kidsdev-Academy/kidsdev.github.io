What exactly is an API?

API stands for Application Programming Interface. That sounds boring, right? Let's use an analogy.

The Restaurant Analogy

Imagine you are at a restaurant. You are the User. The kitchen is the Server (System).

  • You cannot go into the kitchen and cook yourself (Security risk!).
  • You need a messenger to take your order and bring the food.
  • That messenger is the Waiter. The Waiter is the API.

In code, when you use an app like Instagram, your phone (User) sends a request to the API (Waiter), which gets the photo from the Database (Kitchen) and brings it back to you!
