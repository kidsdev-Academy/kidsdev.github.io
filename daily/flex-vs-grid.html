Flexbox vs. Grid

The eternal battle. When should you use which? The answer is simpler than you think: 1 Dimension vs 2 Dimensions.

Flexbox

One Direction Only. Use Flexbox when you want to align items in a single row OR a single column.

1
2
3
style.css
.navbar {
  display: flex;
  gap: 1rem;
}
Navigation Bars Buttons

CSS Grid

Two Dimensions. Use Grid when you need to control rows AND columns at the same time.

1
2
3
style.css
.gallery {
  display: grid;
  grid-template-columns: 1fr 1fr;
}
Photo Gallery Page Layout

The Verdict?

Use Flexbox for components (like buttons inside a menu).
Use Grid for the main page structure.
