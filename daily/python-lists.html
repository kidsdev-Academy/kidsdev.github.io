Python List Magic

Python lists are more powerful than arrays in other languages. You can slice them, reverse them, and step through them with a single line of code.

1. The Reverse Trick

Forget writing a loop. Just use [::-1].

lists.py
numbers = [1, 2, 3, 4, 5]

# Start : Stop : Step
print(numbers[::-1])

>>> [5, 4, 3, 2, 1]

2. Get the Last Item

Use negative numbers to count from the end.

last_item = numbers[-1] // Returns 5
