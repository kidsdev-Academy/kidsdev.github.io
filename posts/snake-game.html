Create Your Own Snake Game in Python

By Nath

Oct 24, 2025 ‚Ä¢ 10 min read

Remember the classic Snake game on old Nokia phones? Today, we are going to build it from scratch! We will use Python and a library called Pygame.

Step 1: Install Pygame

First, we need to install the library that handles graphics and keyboard input. Open your terminal and type:

Terminal
pip install pygame

Step 2: Set up the Screen

Let's create a blank window. Copy this code into a file named snake.py.

snake.py
import pygame
import time

# Initialize the game
pygame.init()

# Define colors
WHITE = (255, 255, 255)
BLACK = (0, 0, 0)
RED = (213, 50, 80)

# Create screen
dis = pygame.display.set_mode((800, 600))
pygame.display.set_caption('KidsDev Snake Game')

# Keep the window open
game_over = False
while not game_over:
  for event in pygame.event.get():
    if event.type == pygame.QUIT:
      game_over = True

  pygame.display.update()

pygame.quit()
quit()

What just happened?

We imported the library, defined some colors (computers see colors as numbers!), and created a "Game Loop". The while not game_over: line keeps the window running until you click the X button.
