How to Build a Discord Bot

In this video, we explore how to connect Python to the Discord API. By the end, you'll have a bot that replies to your messages!

1. Setting up the Environment

First, install the library using your terminal:

Terminal
pip install discord.py

2. The Code

Copy this into your main.py file:

main.py
import discord

client = discord.Client()

@client.event
async def on_ready():
  print(f'Logged in as {client.user}')

client.run('YOUR_TOKEN_HERE')

Security Warning

Never share your bot token! If leaked, regenerate it immediately.
