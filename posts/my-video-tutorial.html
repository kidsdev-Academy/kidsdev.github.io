How to Build a Discord Bot

In this video, we explore how to connect Python to the Discord API. By the end, you'll have a bot that replies to your messages!

1. Setting up the Environment

First, you need to install the library. Open your terminal and type:

pip install discord.py

2. Writing the Code

Here is the basic code structure we used in the video. Copy this into your main.py file.

import discord

client = discord.Client()

@client.event
async def on_ready():
    print(f'We have logged in as {client.user}')

client.run('YOUR_TOKEN_HERE')
