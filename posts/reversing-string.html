Reversing a String Instantly

By Tolani Agbeyangi

Today â€¢ 2 min read

In many languages (like Java or C++), reversing a word requires writing a complex loop. In Python, we have a superpower called Slicing.

The Magic Syntax

You can use square brackets [start:stop:step] to slice text. If you set the step to -1, Python walks backwards through the string.

python_magic.py
word = "KidsDev"

# The [::-1] means "start at end, go to start"
reversed_word = word[::-1]

print(reversed_word)
# Output: veDsdiK

Did you know?

This works on Lists too! [1, 2, 3][::-1] becomes [3, 2, 1].
