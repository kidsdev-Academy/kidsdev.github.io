Design a Superhero Card with CSS

By Nath

Oct 25, 2025 ‚Ä¢ 8 min read

HTML is like the skeleton of a website, but CSS is the costume! Today, we will create a digital ID card for your favorite superhero.

Step 1: The Skeleton (HTML)

We need a container (a div) to hold our hero's name and power.

index.html
<div class="card">
  <h2>Super Coder</h2>
  <p>Power: Infinite Logic</p>
</div>

Step 2: The Costume (CSS)

Right now, it looks boring. Let's add some style! We will change the background color, add rounded corners, and center the text.

style.css
.card {
  background-color: gold;
  padding: 20px;
  border-radius: 15px;
  text-align: center;
  box-shadow: 5px 5px 0px red;
}
