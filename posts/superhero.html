Design a Superhero Card

HTML is the skeleton, but CSS is the costume! Today, we create a digital ID card for a superhero.

Step 1: HTML Skeleton

index.html
<div class="card">
  <h2>Super Coder</h2>
  <p>Power: Infinite Logic</p>
</div>

Step 2: CSS Costume

style.css
.card {
  background-color: orange;
  padding: 20px;
  border-radius: 15px;
  box-shadow: 10px 10px 0px #0F172A;
  color: white;
}
