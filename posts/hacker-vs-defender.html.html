How do websites know if your password is "Strong" or "Weak"? They use Logic! Today, we are going to write a Python script that acts like a security guard (The Defender).

The Logic

We will use an if/else statement. Think of it like a fork in the road:

  • IF the password is too short → Reject it ‚ùå

Open your code editor (like VS Code) and type this out. We will use the len() function to count the letters.

# 1. Ask the user for a password

# 2. Check the length
